# Bem, o que é que temos aqui?

Esse é um caderno de anotações e exemplos. Aqui vou falar sobre matemática, física, data science, software lifecycle e mais temas interessantes, sempre que possível colocando exemplos em python para deixar a teoria mais interessante.

Sempre tive problemas para revisitar as minhas notas de estudo. Adoro escrever em papel, mas fora a questão ambiental, é um parto manter acesso rápido a esse tipo de notas. Para resolver esse problema decidi migrar minhas notas para um padrão digital.

Minhas referências foram notas de aula digitais do professor Fresneda para a disciplina de Algebra Linear na UFABC. Também fui inspirado pelo PDF [CIENCIA DE DADOS](https://www.linkedin.com/feed/update/urn:li:activity:7449916849212805120/) da [LUIZA REIXACH CASTRO](https://www.linkedin.com/in/luizacastro02/).

Meu objetivo é evitar retrabalho reconstruindo notas perdidas através de um repositório versionado git onde essas notas digitais ficarão armazenadas.
